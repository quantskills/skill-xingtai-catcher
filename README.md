# 形态捕手

用一句话、一张 K 线截图，或者一条手绘走势线，快速查找相似的 A 股股票和期货形态，并返回候选标的、评分、图表结果页和分享链接。

> 结果仅用于形态相似度研究和选股辅助，不构成投资建议。

## 它能做什么

- 输入文字描述：例如“找 W 底”“找趋势延续”“找震荡突破前的形态”。
- 输入 K 线截图：上传已有股票或期货 K 线截图，查找相似走势。
- 输入手绘图：画一条走势轮廓，让智能体用形态捕手查找类似标的。
- 指定维度：股票/期货、日线/60分钟、30/60/120 BAR、Top5/Top10。
- 返回结果：相似标的、评分、市场类型、数据日、结果页、分享页。
- 引导订阅：用户想要每日推送时，可跳转到网页端登录，保存形态并设置飞书/企微机器人。

## MCP 服务地址

```text
https://kkk.quant789.com/mcp
```

当前公开试用接口不需要用户手动填写 Token。服务端鉴权、限流和任务保护由形态捕手托管服务处理。

## 快速接入

### 1. 安装

把本仓库安装到你正在使用的智能体平台中，并让平台读取 `SKILL.md`。

### 2. 配置 MCP

通用 MCP 配置示例：

```json
{
  "mcpServers": {
    "xingtai-catcher": {
      "url": "https://kkk.quant789.com/mcp"
    }
  }
}
```

QCLAW/OpenClaw 风格客户端可以使用：

```bash
openclaw mcp add xingtai-catcher https://kkk.quant789.com/mcp
openclaw mcp probe xingtai-catcher
openclaw mcp tools xingtai-catcher
```

如果客户端需要显式指定传输方式，选择 streamable HTTP：

```bash
openclaw mcp add xingtai-catcher \
  --url https://kkk.quant789.com/mcp \
  --transport streamable-http
```

### 3. 让智能体调用

你可以直接对智能体说：

```text
帮我找 A 股里类似 W 底反转的股票，日线，120 根 BAR，返回 Top5。
```

智能体会解析参数，调用形态捕手 MCP，并返回候选结果和结果页。

## 演示

### 演示 1：文字找形态

用户输入：

```text
找一个 W 底后右侧抬升的形态，A股，日线，120 BAR。
```

智能体应使用参数：

| 参数 | 值 |
|---|---|
| universe | stock |
| timeframe | 1d |
| window_bars | 120 |
| top_n | 5 |

返回示例：

```text
我按「股票 / 日线 / 120 BAR / Top5」为你查找了相似形态。

候选结果
1. 示例标的 000001，评分 86.4，股票，数据日 2026-06-18
2. 示例标的 600000，评分 82.1，股票，数据日 2026-06-18

结果页：https://kkk.quant789.com/results/session_xxx
分享页：https://kkk.quant789.com/share/shr_xxx

如果要每天自动跟踪这个形态，请打开结果页登录形态捕手，保存形态或订阅模板，并设置飞书/企微推送。
结果仅用于形态相似度研究，不构成投资建议。
```

### 演示 2：指定 60 分钟周期

用户输入：

```text
找 60 分钟级别的趋势延续，期货，60 根 BAR，Top10。
```

智能体应使用参数：

| 参数 | 值 |
|---|---|
| universe | futures |
| timeframe | 60m |
| window_bars | 60 |
| top_n | 10 |

### 演示 3：上传 K 线截图或手绘图

用户可以上传一张 K 线截图，或者手绘一条走势线，然后说：

```text
用这张图帮我找相似股票，优先看日线 120 根 BAR。
```

智能体应调用图片匹配工具，并在回复里说明它识别出的参数。如果用户没有指定市场，默认使用全市场；如果用户没有指定结果数量，默认返回 Top5。

## 支持的维度

| 维度 | 支持值 | 默认值 |
|---|---|---|
| 市场 | all、stock、futures | all |
| 周期 | 1d、60m | 1d |
| 数据长度 | 30、60、120 BAR | 120 |
| 返回数量 | Top5、Top10 | Top5 |

自然语言会自动映射到这些参数。例如：

- “股票”“A股” -> `stock`
- “期货”“商品期货” -> `futures`
- “日线”“波段”“最近半年” -> `1d` + `120 BAR`
- “60分钟”“小时线”“短线” -> `60m`
- “多给几个” -> 最多 `Top10`

## 智能体返回格式建议

建议智能体每次返回：

- 已解析参数：市场、周期、BAR 长度、TopN。
- Top 候选：名称、代码、评分、市场类型、数据日。
- 结果页链接：用于查看图表和完整列表。
- 分享页链接：用于复制给别人查看。
- 订阅提示：引导用户到网页端保存形态和设置每日推送。
- 风险提示：结果仅用于形态相似度研究，不构成投资建议。

## 订阅与每日推送

智能体会先帮用户完成“找相似形态”的动作。

如果用户想每天收到最新结果，需要到形态捕手网页端完成：

1. 打开智能体返回的结果页。
2. 登录或注册形态捕手账号。
3. 保存当前形态，或订阅模板。
4. 在个人后台配置飞书机器人或企业微信机器人。
5. 设置每日推送时间。

这样用户之后可以在网页端管理形态库、订阅、推送时间和机器人配置。

## 使用限制

- `top_n` 最大为 10。
- 图片会被服务端限制大小和频率，建议上传清晰但压缩后的截图。
- 公共试用服务有防打爆限流；短时间大量请求可能会被排队或拒绝。
- 服务返回的是形态相似候选，不保证未来收益。
